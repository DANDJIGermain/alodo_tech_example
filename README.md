# ALODO TECH — Digital Infrastructure for Emerging Markets

ALODO TECH est un bâtisseur d'infrastructures digitales conçu pour connecter l'économie informelle africaine au système financier mondial. Cette plateforme Next.js sert de vitrine institutionnelle et technologique pour présenter nos solutions d'inclusion financière, d'IoT et d'intelligence artificielle.

## 🚀 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (Modern Custom Properties & Design Tokens)
- **Animations**: CSS Transitions & Intersection Observer API
- **Icons**: SVG optimisés
- **Deployment**: [Vercel](https://vercel.com)

## 🛠️ Architecture du Projet

```text
/src
  /app           # Routes, pages et styles globaux
  /components    # Composants réutilisables (Navbar, Footer, etc.)
/public
  /partenaires   # Assets et logos partenaires
  /products      # Visuels produits
```

## ⚙️ Installation & Développement

1. **Cloner le projet** :
   ```bash
   git clone git@github.com:DANDJIGermain/alodo_tech_example.git
   cd ATAPP
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```
   L'application sera disponible sur `http://localhost:3000`.

## 🌐 Localisation (FR/EN)

Le projet intègre un système de gestion de langue dynamique géré via le contexte React. Les traductions sont centralisées dans les composants pour une maintenance facilitée.

## 🏗️ Principes de Design

- **Premium Identity** : Utilisation du Glassmorphism, de typographies modernes (Inter, JetBrains Mono) et de micro-animations.
- **Responsiveness First** : Entièrement optimisé pour tous les écrans, du mobile au desktop 4K.
- **Performance** : Optimisation des images et des assets pour un chargement rapide sur les réseaux mobiles africains.
